本文可包含 %(attribute)s 格式字串，其可替換為通信論壇 相應的屬性。詳情請參照 Python 格式化字串規則. 預設屬性如下：
  • real_name - 本論壇 "較佳" 的名稱； 通常都用大寫。
  • list_name - 此論壇名稱用於 URL 識別 ，其大小寫不可有誤。
  • host_name - 執行論壇的伺服器詳盡的主機名稱（FQDN）。
  • web_page_url - Mailman 的 URL base. 其後可添加 一些參數以產生有用的網址。 例如： 加上 listinfo/%(list_name)s 可產生通信論壇的 listinfo 網址。
  • description - 該通信論壇的摘要說明。
  • info - 該通信論壇的詳細說明。
